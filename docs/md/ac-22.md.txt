# Publicly Accessible Content
---
## Descriptive
How and when and by whom the set of data was collected.



Metadata describes how and when and by whom a particular set of data was collected, and how the data is formatted.

## Structural 
How the data is formatted.

The data source set is JSON (JavaScript Object Notation) and follows the formatting for ***object*** ```ac-22``` followed by an ***array*** of values.
  
```
{
  "ac-22": [
    {
      "title": "",
      "honorific": "",
      "firstname": "",
      "lastname": "",
      "training": "",
      "date": ""
    }
  ]
}
```
